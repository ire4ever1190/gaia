Why Local Beats Cloud

Four differentiators vs. Superhuman, Shortwave, Fyxer, and Microsoft Copilot.

🔒

Local Inference

Email content never leaves the device. Cloud triage services process your inbox on their servers — every message, every attachment, every thread. GAIA runs Gemma 4 on the Ryzen AI NPU. Zero outbound network calls with email content, verified by audit log.

🎯

Per-Cohort Autonomy

Not one global dial. Users set autonomy per sender cohort: L5 for newsletters (auto-archive), L2 for colleagues (suggest, don’t act). 8 cohorts × 6 autonomy levels = fine-grained control no cloud product offers.

🔍

Auto-Discovered Integration

Agent detects installed email clients via OS signals (Win registry, macOS defaults, Linux xdg), infers provider from domain + MX records, and walks through OAuth. User enters email address — everything else is automatic.

📖

Auditable by Construction

Every action is logged, reversible at L4+, and the agent code is open-source. Users can inspect exactly what the agent remembers, what it categorized, and why. No black-box cloud decisions.

Privacy & Security

Critical Policy

READ Autonomous — agent reads freely
SEND Always requires user confirmation
ARCHIVE Opt-in only
DELETE Always confirms first

All email processed on-device. Credentials in encrypted vault. Only summaries stored — never full email bodies. Every action logged, reversible at L4+, code is open-source.
